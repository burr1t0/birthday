# 🐱 Birthday Invite

Сайт-приглашение на день рождения с персональными кодами и общим виш-листом.

---

## 📝 Что редактировать

**Только один файл:** `guests.js`

| Раздел | Что заполнить |
|--------|---------------|
| `GUESTS` | Код, имя, эмодзи и текст приглашения для каждого гостя |
| `PARTY` | Имя именинницы, дата, время, место, адрес |
| `WISHLIST_ITEMS` | Названия подарков и ссылки (ссылки необязательны) |

> ⚠️ **id** у пунктов виш-листа (`w1`, `w2`…) **не меняй** — это ключи в Firebase.

---

## 🔥 Настройка Firebase (для виш-листа)

> Без Firebase виш-лист работает, но только локально — гости не видят чужие пометки.

1. Зайди на [console.firebase.google.com](https://console.firebase.google.com)
2. **Создать проект** (название любое)
3. Левое меню → **Realtime Database** → **Создать базу данных** → Выбери регион → **Тестовый режим** ✓
4. ⚙️ Настройки проекта → **Ваши приложения** → значок `</>` → введи имя → **Зарегистрировать**
5. Скопируй объект `firebaseConfig`
6. Открой `wishlist.html`, найди блок **`🔥 FIREBASE CONFIG`**, вставь свои данные

---

## 🚀 Деплой на Netlify (бесплатно, 2 минуты)

1. Зайди на [app.netlify.com](https://app.netlify.com) → войди через GitHub или email
2. **Add new site → Deploy manually**
3. Перетащи папку `birthday-invite` прямо в окно браузера
4. Netlify даст ссылку вида `https://random-name.netlify.app` — это и есть твой сайт!

Ссылку можно переименовать: **Site configuration → Change site name**.

## 🐾 Коды гостей

Коды вводятся в **любом регистре** — сайт автоматически приводит к верхнему.

Пример кода по умолчанию: `DASHA-001`

Можно использовать любой формат: `МАША`, `CAT42`, `BESTIE-2025` и т.д.
